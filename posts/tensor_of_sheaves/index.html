Tensor of sheaves

Nov 05, 2018
4 min read

Following the Excersice 2.5.13 from Justin Curry's thesis we show that, in general, tensor of sheaves is not a sheaf. We begin by proving that the set of sections of a continouos function and its free vector space give a sheaf. Finally, we show that if we take $f(z)=z^2$, from $S^1$ to $S^1$, the tensor of its section vector spaces is not a sheaf.

Let $f:Y\rightarrow X$ be a continouos function, for every subset $U\subset X$ we denote the set of section of $f$ on $U$ by

\[ Q(U) = \left\{ s \colon U \rightarrow Y \text{ continuous }\mid f \circ s (x) = x, \ {\rm for} \ {\rm all} \ x \in U \right\}. \]

We want to check that the assignement \(Q\colon {\rm Open}(X)^{\rm \bf op} \rightarrow {\rm \bf Set}\) defines a sheaf. First lets observe that the codomain is indeed the category of sets, if $V\rightarrow U$ in ${\rm Open}(X)^{\rm \bf op}$, i.e. $U\subset V$, then $Q(i)(s) = s|_U$ is a section from $f$ restricted to $U$.

Lets verify that it is a sheaf, we take an open set $U \subset X$ and a cover \(\mathcal{U} = \{U_i \}_{i \in I}\) of $U$, where $I$ has a strict total order. We need to show that the following sequence is an equalizer:

\[ \xymatrix{ Q(U) \ar[r]^(.4){e} & \prod \limits_{i \in I} Q(U_i) \ar[r]^(.45){f^- f^+} & \prod \limits_{\substack{i, j \in I \\ i < j}}Q(U_i \cap U_j) } \]

where \(e(s)_i=s|_{U_i}\), \(f^-_{ij} = \rho_{ij,i}\circ \pi_i\) and \(f^+_{ij}=\rho_{ij,j}\circ \pi_j\). Let \(e':W\rightarrow \prod_{i\in I}^{} Q(U_i)\) such that \(f^-\circ e' = f^+\circ e'\). Then we define the map $\mu:W \rightarrow Q(U)$ by \(\mu(w)(x) = e'(w)_i(x)\) where $x\in U_i$. Let see that it doesn't depend on the choice of $i$, if $x\in U_j$ then

\[ \begin{align*} e'(w)_i(x) &= (\pi_i \circ e'(w))(x) \\ &= (\pi_i \circ e'(w))|_{U_i\cap U_j}(x) \text{ as } x\in U_i\cap U_j\\ &= (\rho_{ij,i}\circ \pi_i\circ e'(w))(x) \\ &= f^-_{ij}\circ e'(w)(x) \end{align*} \]

and analogously we see that \(e'(w)_j(x) = f^+_{ij} \circ e'(w)(x)\), so it well defined and $e' = e \circ \mu$. This proves that $Q(U)$ is an equalizer, so $Q$ is a sheaf.

In particular consider $f:S^1\rightarrow S^1$ given by $f(z)=z^2$, notice that $Q(S^1) = \emptyset$ since otherwise there would be a section $s\colon S^1 \rightarrow S^1$ such that $s(z)^2 = z$ for all $z \in S^1$. In particular

\[ \xymatrix{ \mathbb{Z} = \pi_1(S^1) \ar[r]^{\times 2} & \pi_1(S^1) = \mathbb{Z} } \]

would be a surjective map, but we know that this is not true since $1 \notin 2\mathbb{Z}$.

Now consider the assignement $Q_k^\otimes$ given on open sets $U$ of $X$ by

\[ \xymatrix{ U \ar@{|~>}[r] & Q_k(U) \otimes_k Q_k(U), } \]

where $Q_k(U)$ is the sheaf that assigns to an open set $U$ the free vector space generated by the set $Q(U)$, we want to show that this is not a sheaf. Notice that since the tensor product $\otimes_k$ is defined over $k$, one can define it naturally by taking the dot product $\cdot_k$ between sections in $Q_k(U)$. In particular, whenever we take a section $s \in Q_k(U)$, we have the property

\[ s(z)\cdot_k s(z) = s(z)^2 = (f \circ s)(z) = z. \]

Next, take the cover of $S^1$ given by \( U = {\rm exp}(i(-\varepsilon, \pi + \varepsilon)) \ \ \ \ { \rm and} \ \ \ \ V = {\rm exp}(i(\pi -\varepsilon, 2\pi + \varepsilon)). \) It is clear that there is a section $s_U \in Q_k(U)$, e.g. we can take $s_U(z) = (z)^{1/2}$, and by the argument above \(s_U \cdot s_U = {\rm Id}_{|U}\). Similarly, there is a section \(s_V \in Q_k(V)\) satisfying \(s_V \cdot s_V = {\rm Id}_{|V}\). Thus we have that \({\rm Id}_{|U} \in Q_k^\otimes(U)\) and \({\rm Id}_{|V} \in Q_k^\otimes(V)\), and their restrictions to \(Q_k^\otimes(U\cap V)\) are equal \(({\rm Id}_{|U})_{|V} = {\rm Id}_{|U\cap V}= ({\rm Id}_{|V})_{|U}\). Thus, if $Q_k^\otimes$ was a sheaf, there would exist an element in

\[ Q_k^\otimes(U \cup V) = Q_k(U\cup V) \otimes Q_k(U\cup V) = 0 \otimes 0 = 0 \]

extending both \({\rm Id}_{|U}\) and \({\rm Id}_{|V}\), but this does not make sense.
